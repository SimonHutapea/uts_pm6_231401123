## GoQuiz

GoQuiz adalah aplikasi kuis yang dibuat menggunakan bahasa pemogramman Flutter. Aplikasi ini memiliki tiga mode kuis berbeda yaitu, Classic, Timer, dan Endless. Kamu akan diminta untuk mengisi nama terlebih dahulu sebelum memulai kuis. Hasil kuis akan ditampilkan di Leaderboard pada akhir kuis, kamu juga dapat melihat seluruh Leaderboard dari Beranda.
